# AHK输入法键盘布局自动切换

#### Description
基于AutoHotkey的自动切换输入法键盘布局。
1.可针对不同的应用自动切换中英文键盘布局
2.可设置切换输入法在屏幕右下角文字提示
3.可设置切换输入法在鼠标位置文字提示
4.可设置中英文键盘布局、提示方式、文字提示样式及内容、应用默认输入法等
